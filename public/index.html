Developer-first

A full API for everything

Create meetings, manage participants, and build automations with a clean REST API and Socket.IO real-time events. Integrate in minutes.

# Create a meeting in one line
curl -X POST https://meetingservice.app/api/meetings \
  -H "x-api-key: mk_your_key" \
  -d '{"title": "Team Standup"}'

# Response
{
  "meetingId": "abc-defg-hij",
  "adminToken": "uuid-token",
  "joinUrl": "/join/abc-defg-hij"
}

# Socket.IO (Node.js)
const socket = io("https://meetingservice.app");
socket.emit("join-meeting", {
  meetingId: "abc-defg-hij",
  name: "My Bot"
});
